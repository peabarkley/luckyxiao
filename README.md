# LuckyXiao Zodiac IP Official Website

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This is the official website for **LuckyXiao Zodiac IP**, a creative brand that reimagines the traditional Chinese zodiac with modern,个性 personalities. The website showcases 12 zodiac characters, official merchandise, and the brand's core philosophy "BORN TO BE DIFFERENT".

## ✨ Features

- **Bilingual Support**: Full Chinese and English versions available
- **12 Zodiac Characters**: Each zodiac has its own detailed profile page with personality traits and special skills
- **Official Merchandise**: Display of blind boxes, T-shirts, plush dolls, and canvas bags
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **No Dependencies**: Pure HTML/CSS/JavaScript, no frameworks required
- **Easy to Customize**: Simple file structure, all content can be modified easily
